# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-05-21

### Added

- Expand Manhattan map (avenues, plazas, landmarks, street labels, minimap, snow)
- Tree inventory by species (balsam, Douglas, Fraser, spruce, noble) with wholesale restock
- Lot upgrades (heater, neon sign, lights, cider stand), rent days, and day-phase price multipliers
- Ten customer types with expanded multi-round dialog; skepticism meter instead of hearts
- Ambient NPC pedestrians, global chat (T), tutorial tips, and improved HUD

### Changed

- Vendor encounters use multi-round dialog; reputation and earnings track total career stats
- Larger canvas, stand outlines, and interaction prompts at lots

## [0.1.1] - 2026-05-21

### Changed

- Document static landing URL at **`/xmas-game/`** on hromp.com

## [0.1.0] - 2026-05-21

### Added

- Initial release: top-down NYC tree-vendor game with login, multiplayer, Pokemon-style sales battles, multiple stands, and Romp Family Christmas Trees as the prestige final stand after three good seasons.
